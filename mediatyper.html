Media-Typer

Media typer står for hvordan innhold i en webside blir sett fra ulike enheter.
Dette kan være PC, mobil, printer, Tv, skjermlesere, selv ulike programvarer.
I CSS finnes det flere Mediatyper som er mulig å legge inn slik at websiden skal respondere ut ifra hvilken enhet man ser den på tilsvarende. All: gjelder for alle enheter.
Aural: gjelder for skjermlesere.
Braille: for taktile tilbakemeldingslesere.
Embossed: for printere som kan skrive ut i braille format.
Handheld: gjelder mobiler og padder som kan ha touch-funksjoner osv.
Print: for utskriftsversjon, samt utskrift ved bruk av printer.
Projection: for visning via prosjektorer.
Screen: for data skjermer.
Tty: for medier som bruker lyd som terminaler og teletypes.
Tv: for TV enheter.
Det er mulig å spesifisere mediatyper gjennom CSS på ulike måter.

CSS-Layouts

CSS GRID: CSS grid brukes som et layout oppsett i CSS for å fraksjonere siden i rader og kolonner.
Med grid kan man enkelt lage websider som er responsive, samt bestemme hvor innhold skal plasseres.
Det finnes forskjellige måter for å plassere grid elementer. Den enkleste er å bruke «fr» (fractions) eller px(pixels) til å bestemme størrelsen på selve Grid område.

Grid har flere egenskaper:
element plassering: kan plassere elementer i spesifikke områder ved å bruke grid kolonner og rader. Grid har også en innebygd algoritme som kontrollerer plasseringen av elementer som ikke har fått en spesifikk plassering. Ut fra disse verdiene kan man da enkelt plassere og flytte elementer:
Column i Grid står for kolonner så disse vil gå fra høyre mot venstre og dekke hele siden. Disse går vertikalt. Grid-template-columns:1fr 1fr; (Deler siden opp i 2 like kolonner.)
Rows i grid deler opp selve radene nedover disse går horisontalt.
Grid-template-row: 1 fr 2fr 3fr ; ( deler siden opp i 3 seksjoner hvor første rad er 1 brøkdel, så 2, så 3). Ved bruk av repeat() element er det mulig å repetere et ønsket oppsett ut ifra grid elementene.

Grid-gap: brukes til å lage et mellomrom istedenfor å bruke margin rundt det valgte grid området. Dette gjør det enkelt å få et fint og presist mellomrom mellom alle bokser og elementer innenfor grid området. Det er også mulig å neste en grid inne i en grid, ved å lage en container inne i en container og så legge til elementer, for å splitte opp en større boks.

Flexbox har samme funksjon som grid men distribuerer plass rundt en enkel kolonne eller rad, kan sammenliknes med float, men mye bedre.
Flexbox brukes på lik linje som grid.
Oppsettet for flexbox er ganske lik Grid. Det brukes en container hvor diverse item elementer blir satt inn. Disse refereres som parent og child.
Containeren holder flex elementer, denne boxen kan plasseres både horisontalt eller vertikalt hovedaksen.

Flexbox sine egenskaper: Det finnes mange egenskaper i flexbox. display:flex; - gir det valgte elemente (parent og child) flex egenskapen. Dersom denne ikke er med i container elementet vil ikke flex-egenskaper fungere.
Flexbox-wrap brukes dersom man ønsker at child elementene skal følge under hverandre dersom skjermen blir mindre.

Samtidig er denne layoutmetoden mye enklere å bruke enn å måtte presisere alle egenskaper som ellers måtte til for å få tilsvarende responsivitet.
